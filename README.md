## 📖 Project Overview

This project analyzes pizza sales data using **SQL Server** and **Microsoft Excel** to uncover valuable business insights and track key performance indicators (KPIs).

The goal is to help stakeholders understand sales performance, customer ordering patterns, product popularity, and revenue contribution across categories and sizes.

---

## 🎯 Business Problem

A pizza restaurant wants to understand:

- Which pizzas generate the most revenue?
- What are the busiest days and hours?
- Which pizza categories perform best?
- Which pizza sizes contribute the most sales?
- Which products should be promoted or improved?

This analysis answers these questions using SQL queries and an interactive Excel dashboard.

## 📌 Key Insights

### 🏆 Best Performing Category
Classic Pizza category contributes the highest sales and order volume.

### 📏 Most Popular Size
Large pizzas generate the maximum revenue contribution.

### 📅 Busiest Day
Friday records the highest number of orders.

### ⏰ Peak Ordering Hours
Customer activity is highest during:
- 12 PM – 1 PM
- 4 PM – 8 PM

### 🍕 Best Selling Pizza
Classic Deluxe Pizza is among the top-performing products.

### 📉 Lowest Selling Pizza
The Brie Carre Pizza generates the lowest sales volume.
